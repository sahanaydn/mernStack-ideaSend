# mernStack-ideaSend

MERN stack idea send 

**First you need to create a database with mongodb atlas and paste your url in the relevant place in the server.js folder.**

SERVER

$ cd api-idea

$ npm install

Client

$ cd frontend-idea

$ npm install


RUN & START

Server

$ cd api-idea

$ nodemon server.js

Client

$ cd frontend-idea

$ npm start
